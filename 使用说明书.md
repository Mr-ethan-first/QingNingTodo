# 青柠待办 · 使用说明书（Windows 版）

欢迎使用「青柠待办」——一款帮你**规划、专注、记录、复盘**的本地时间管理工具。
它基于番茄工作法，完全在你的电脑上运行，数据保存在本地，不联网、无广告。

---

## 一、第一次打开

- 双击 `青柠待办.exe` 即可启动（默认使用本地存储，无需任何配置）。
- 如果你拿到的是源代码：安装好 Python 后，在文件夹里打开命令行，输入：

  ```powershell
  pip install -r requirements.txt
  python -m src.main
  ```

启动后，左侧是一排导航按钮，右侧是对应页面的内容。

---

## 二、主界面与导航

窗口左侧的导航栏包含 5 个主要页面：

| 导航 | 用来做什么 |
| --- | --- |
| **待办清单** | 管理你今天和以后的所有任务 |
| **专注计时** | 开始番茄钟、正计时、倒计时，专注做事 |
| **数据统计** | 查看你的专注成果与习惯趋势 |
| **未来计划** | 记录重要日子，看看还有多久到来 |
| **设置** | 个性化外观、计时参数、数据管理等 |

右上角常驻「新建待办」「待办集」等快捷按钮。

### 窗口操作小贴士
- **最小化 / 收起**：点击标题栏左侧的「最小化」按钮，或点击任务栏上的「青柠待办」图标，窗口会平滑缩到任务栏收起；再次点击任务栏或右下角托盘图标即可展开。
- **关闭**：点击标题栏「关闭」会弹出确认框，可选择「退出程序」或「最小化到托盘」（后台常驻，双击托盘图标可再唤出）；勾选「不再提醒」后下次直接退出。
- **托盘常驻**：收起到托盘后程序仍在后台运行，专注计时、习惯提醒、白噪音等都不受影响。

## 三、待办清单

### 1. 新建待办
点击右上角 **「新建待办」**，填写：
- **标题**：比如"写周报""背单词"。
- **计时方式**：普通番茄钟 / 正计时 / 倒计时 / 严格模式。
- **时长**：预计投入多少分钟。
- **分组（待办集）**：归入某个主题，比如"工作""学习"。
- **优先级**：普通 / 重要 / 紧急。
- 还可设置重复、提醒时间等。

### 2. 待办集分组
点击右上角 **「待办集」** 新建分组（如"工作""生活"）。待办可归入不同待办集，顶部下拉框可快速筛选。

### 3. 养习惯与打卡
把待办类型设为"养习惯"，设定单位与目标（如"每天读书 30 分钟"）。完成后在待办上点 **「打卡」** 记录，坚持看得见。

### 4. 调整顺序
直接用鼠标**拖动**卡片上下排序；也可以右键卡片 → **「排序｜移动」** 选择置顶、上移、下移，或移动到其他待办集。

### 5. 更多设置（右键卡片）
- **完成后第二天不再显示**：完成的待办次日自动隐藏，保持清单清爽。
- **此待办始终关闭学霸模式**：该任务专注时不启用学霸模式。
- **循环计时**：一个任务连续多轮番茄钟。
- **自定义休息时长**：仅为这个待办设置专属休息时间。

### 6. 更换背景（右键卡片 → 更换背景）
每个待办都能拥有**自己的背景图**，卡片会变成通透的透明风格，好看又个性化。

### 7. 定时提醒（右键卡片）
为待办设置提醒时间，到点不错过。

### 8. 双击进入专注
**在待办卡片上双击**，即可直接进入「专注计时」页面并开始该待办的专注，任务与计时自动关联。

### 9. 其它右键操作
开始专注、编辑、标记完成/进行中、更多设置、更换背景、定时提醒、打卡（养习惯）、专注历史记录、排序移动、删除。

## 四、专注计时

在「专注计时」页面点 **开始**，即可进入专注：

- **暂停 / 继续 / 放弃 / 完成**：随时控制节奏。
- **休息**：一轮结束后可休息，到点自动提醒回来。
- **白噪音（背景音乐）**：内置多种环境音（雨声、海浪、轻音乐等），帮你快速进入心流；也支持自定义上传。
- **背景海报**：在「设置 → 外观｜主题 → 背景海报」为专注页选一张喜欢的图片。
- **学霸模式**：开启后专注时仅允许白名单里的程序运行，帮你远离干扰（白名单在设置中管理）。

> 小贴士：在「待办清单」里**双击待办**就能直接带着这个任务开始专注。

## 五、数据统计

「数据统计」用图表展示你的专注成果：

- **概览卡片**：累计专注、今日专注等关键数字。
- **时长分布**：柱状图 + 饼图，看时间花在哪类任务上。
- **时段热力图**：发现你一天里最高效的时间段。
- **月度趋势**：坚持情况的曲线。
- **起床 / 睡眠分布**：了解自己的作息规律。
- **打断原因**：记录并复盘专注时被什么打断了。
- **年度数据**：回望一整年的积累。
- 还可查看每条**专注记录**，并在「统计设置」里调整图表单位与展示方式。

## 六、未来计划

记录重要的日子（考试、旅行、生日……），页面会显示**距离今天还有多久**。即使日子过去也不会消失，留作纪念。

## 七、设置

点击左侧 **「设置」** 进行个性化：

- **个人信息**：设置你的昵称。
- **外观｜主题**：切换 **皓白（清新白）** 与 **墨夜（深邃黑）** 两套主题，还能自定义主题色；设置专注页背景海报。
- **专注计时设置**：学霸模式开关、严格模式、倒计时自动转正计时、番茄钟格言、自定义休息/暂停上限、完成提示音、白噪音、进入休息前是否询问。
- **待办显示设置**：午夜模式（跨天归属）、固定排序、完成的待办是否划线、记住待办集展开状态、待办较多时显示搜索框等。
- **习惯提醒**：设置习惯未完成的提醒时间。
- **系统集成**：开机自启、全局快捷键（随时开始/暂停专注）。
- **习惯养成**：设定长期目标（日/周/月目标时长）。
- **定时锁屏**：配置锁屏时段（桌面行为管控，可按需扩展）。
- **本地数据**：查看数据存储位置、清除全部数据、切换数据库后端（默认本地，可选 MySQL 用于多设备同步）。
- **关于｜帮助**：查看版本，打开本使用说明书。

## 八、常见问题

**Q：我的数据存在哪里？**
A：默认保存在电脑用户目录下的 `.qingning_todo` 文件夹里（本地文件），不会上传到任何网络。

**Q：如何备份或迁移数据？**
A：复制该 `.qingning_todo` 文件夹即可。如需在多台电脑间同步，可在「设置 → 本地数据 → 数据库后端」切换为 MySQL。

**Q：换电脑后背景图不显示了？**
A：背景图按你选择的文件路径保存，建议把图片放在固定位置，或重新为该待办"更换背景"。

**Q：不小心删了待办能找回吗？**
A：删除后在「设置 → 本地数据」可清除数据，但已删除的单项暂不支持撤销，请谨慎操作。

**Q：双击待办没反应？**
A：请用鼠标**左键双击**待办卡片；若仍无反应，确认不是单击拖拽。
