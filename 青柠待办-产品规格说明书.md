## 二、产品概述

青柠待办是一款基于「番茄工作法（Pomodoro Technique）」的本地时间管理与专注效率工具，核心理念为 **规划 → 追踪 → 记录 → 复盘**。

产品三大核心模块：

1. **待办清单**：规划每一天的任务，让一天在计划中有条不紊。
2. **番茄钟计时**：以 25 分钟工作 + 5 分钟休息为基础的可自定义专注计时。
3. **数据统计**：以图形化方式直观展示专注成果，提供成就感与自我复盘依据。

设计语言「一抹青柠」：

- 以清新青柠绿（`#7CB342` 浅色 / `#AED581` 深色）作为主色，配合冷调副强调色（时段 / 连续天数等次级数据）。
- Win11 Fluent 风格：大圆角卡片 + 分层投影、半透明玻璃质感侧栏、选中态左侧高亮竖条 + 柔和药丸底色。
- 提供「皓白 · 清新白」与「墨夜 · 深邃黑」两套主题，并支持自定义主色调；切换主题时全局 QSS 与自绘控件（图表 / 图标）配色统一重建。

> 产品定位为：**简约高效的「番茄钟 + 待办列表 + 数据统计」桌面工具，无广告、秒启动、纯本地存储。**

## 四、详细功能规格

### 4.1 待办清单（TodoList）

#### 4.1.1 功能描述
待办清单是产品入口页，展示用户的所有待办事项卡片；点击卡片菜单中的「开始专注」即进入专注计时页。

#### 4.1.2 功能明细

| 编号 | 功能点 | 规格说明 |
| --- | --- | --- |
| T-01 | 新建待办 | 右上角「新建待办」打开对话框，支持「普通番茄钟 / 养习惯 / 定目标」三种类型（定目标为占位） |
| T-02 | 待办集（分类） | 可创建待办集（分组），用于归类待办；分组可选颜色标记 |
| T-03 | 计时方式选择 | 普通番茄钟 / 正计时 / 倒计时 / 严格模式（养习惯类型下为 倒计时 / 正计时 / 不计时） |
| T-04 | 时长设置 | 可自定义专注时长（分钟）、休息时长（分钟）、循环次数 |
| T-05 | 重复选项 | 不重复 / 每日 / 每周 / 每月（数据层支持，按重复规则生成周期任务） |
| T-06 | 定时提醒 | 可为待办设置每日提醒时间（本地配置，由习惯提醒时间等机制辅助） |
| T-07 | 更换背景 | 可为待办卡片选择本地背景图片 |
| T-08 | 编辑 / 删除 | 卡片菜单提供编辑、删除（删除有二次确认） |
| T-09 | 开始专注 | 卡片菜单「开始专注」跳转专注页并载入该待办 |
| T-10 | 完成与勾选 | 可标记完成 / 进行中，完成态显示删除线（可设置关闭） |
| T-11 | 优先级 | 普通 / 重要 / 紧急，参与排序与展示 |
| T-12 | 排序与移动 | 支持拖拽排序，菜单提供「置顶 / 上移 / 下移 / 移动到分组」 |
| T-13 | 更多设置 | 单待办维度：完成后次日不再显示、此待办始终关闭学霸模式、循环计时、自定义此待办休息时长 |
| T-14 | 养习惯打卡 | 养习惯类型待办提供「打卡」入口，按日去重记录 |
| T-15 | 专注历史记录 | 查看某待办关联的专注记录（日期 / 时长 / 完成状态） |
| T-16 | 列表过滤 | 按待办集 + 状态（进行中 / 已完成 / 全部）过滤 |

#### 4.1.3 计时方式定义

| 计时方式 | 定义 |
| --- | --- |
| 普通番茄钟 | 倒计时工作（默认 25 分钟），结束进入休息流程 |
| 正计时 | 从 0 开始累计，手动结束 |
| 倒计时 | 设定固定时长倒数计时 |
| 严格模式 | 计时开启后不可暂停（触碰暂停即禁用），用于强约束专注 |
| 不计时（养习惯） | 养习惯类型专用，仅记录打卡，不进入倒计时 |

### 4.2 番茄钟（专注计时核心）

#### 4.2.1 功能描述
专注计时页是核心计时引擎。从待办进入或手动选择待办（无待办时提供默认 25 分钟番茄钟）后开始计时，支持循环、休息、放弃与提取完成。

#### 4.2.2 专注界面控件

| 编号 | 控件 | 规格说明 |
| --- | --- | --- |
| P-01 | 开始 / 暂停 / 继续 | 开始计时；运行中可暂停（严格模式不可用，显示「继续」恢复） |
| P-02 | 提取完成 | 结束并保存本次专注记录（区别于「放弃」） |
| P-03 | 放弃 | 放弃当前计时，可填写打断原因（不计入有效专注） |
| P-04 | 循环轮次 | 顶部显示「第 X/Y 轮」，按设定循环次数自动轮转 |
| P-05 | 白噪音选择 | 计时页内直接选择环境音（关闭 / 自然音 / 氛围音 / 轻音乐 三大类），支持自定义上传 |
| P-06 | 自定义格言 | 计时器下方展示激励文字（可在设置中修改） |
| P-07 | 选择待办 | 无载入待办时，弹出选择器挑选待办或默认番茄钟 |

#### 4.2.3 休息模式

| 编号 | 功能点 | 规格说明 |
| --- | --- | --- |
| R-01 | 休息前询问 | 专注完成后弹窗询问是否进入休息（可跳过 / 放弃本次） |
| R-02 | 休息时长 | 默认 5 分钟（全局可改），并支持单待办自定义休息时长 |
| R-03 | 跳过 / 提前结束 | 可跳过休息直接开始下一轮，或提前结束休息 |
| R-04 | 自动轮转 | 有下一轮时自动开始下一轮专注；全部完成提示「全部完成」 |

#### 4.2.4 番茄钟高级特性

| 编号 | 功能点 | 规格说明 |
| --- | --- | --- |
| P-08 | 自定义番茄 / 休息 / 循环 | 时长、休息、循环次数均可自定义（全局 + 单待办覆盖） |
| P-09 | 倒计时自动转正计时 | 倒计时归零后，开启开关可自动切换为正计时继续累计记录 |
| P-10 | 暂停时间上限 | 暂停超过上限（默认 3 分钟，可设 0 为不限制）自动放弃本次专注 |
| P-11 | 午夜模式 | 凌晨 0–4 点发起的计时，归属统计到前一天（跨零点处理） |
| P-12 | 打断原因记录 | 放弃时填写原因，写入专注记录并可统计月度打断分布 |

> 说明：专注页「锁屏番茄钟」「悬浮窗」等能力在当前版本未实现，相关入口已在 3.2 / 3.3 标注。

### 4.3 专注模式（学霸模式，Windows 本地版）

#### 4.3.1 功能描述
原「学霸模式」在移动端于专注期间锁定娱乐 App、拦截消息。Windows 本地版将其适配为 **桌面专注模式**，当前版本提供「开关 + 白名单」的数据管理能力。

#### 4.3.2 功能明细

| 编号 | 功能点 | 规格说明 |
| --- | --- | --- |
| F-01 | 学霸模式开关 | 设置中一键开启 / 关闭；开启后展开白名单管理 |
| F-02 | 白名单管理 | 添加允许例外的应用（应用名称 + 进程名，如 `chrome.exe`），可删除 |
| F-03 | 单待办豁免 | 可在待办「更多设置」中设置「此待办始终关闭学霸模式」 |
| F-04 | 严格模式联动 | 严格模式开启时，计时不可暂停，强化约束 |

> 平台适配说明：系统级「拦截非白名单窗口 / 通知勿扰」行为当前版本尚未接入，需后续结合 Windows 前台窗口监控、专注助手（勿扰）或进程管控扩展（参见 3.2）。

### 4.5 白噪音

#### 4.5.1 功能描述
专注期间可播放环境音帮助进入状态。当前版本内置约 38 种真实音频素材（已预转换为 WAV 格式），按内容分为三大类，通过 Windows `winsound` 原生循环播放；也支持用户自定义上传音频（上传时自动转码为 WAV）。

#### 4.5.2 功能明细

| 编号 | 功能点 | 规格说明 |
| --- | --- | --- |
| W-01 | 白噪音库 | 内置约 38 种真实音频，分三大类：**自然音**（雨声 / 海浪 / 森林鸟鸣 / 雷声 / 风声 / 溪流 等）、**氛围音**（咖啡厅 / 篝火 / 蟋蟀合唱 / 时钟嘀嗒 / 白噪音 等）、**轻音乐**（钢琴 / 冥想 / 氛围 等）；素材来自免费资源库，统一为 WAV 格式 |
| W-02 | 专注播放 | 专注计时页或设置中可选环境音，专注期间播放 |
| W-03 | 循环播放 | 基于 `winsound` 的 `SND_LOOP` 标志实现原生无缝循环 |
| W-04 | 与计时联动 | 开始 / 停止计时时随之播放 / 停止；可配置「计时开始自动播放白噪音」(`auto_play_on_start`) |
| W-05 | 自定义上传 | 用户可上传本地音频，自动转码为 WAV 后加入播放列表（仅非内置条目可删除） |
| W-06 | 完成提示音 | 计时完成提示音可选「系统默认 / 无」 |
| W-07 | 开关与独立播放 | 设置中「开启背景音（白噪音）」(`bg_music_enabled`) 控制是否启用环境音；播放器为进程级单例，避免多实例产生孤儿音轨 |

> 说明：当前版本未提供白噪音音量控制滑块，使用系统默认播放音量（参见 3.3）。非 Windows 平台优雅降级（仅提示，不发声）。

#### 4.6.2 统计指标与图表

| 编号 | 组件 | 说明 |
| --- | --- | --- |
| S-01 | 概览卡片 | 累计次数 / 累计时长 / 日均时长；今日次数 / 今日时长 / 今日放弃次数 |
| S-02 | 专注时长分布 | 日 / 周 / 月 / 自定 四档切换 + 柱状图 + 按待办占比饼图 |
| S-03 | 本月专注时段分布 | 日 × 小时 热力图，直观呈现每天各时段专注强度 |
| S-04 | 月度数据 | 面积 / 折线趋势图（可显示近 7 天或整月） |
| S-05 | 起床 / 睡眠打卡分布 | 按月统计起床、睡眠打卡的时段分布柱状图 |
| S-06 | 月度打断原因分布 | 饼图展示当月各打断原因占比 |
| S-07 | 年度数据 | 12 个月专注时长趋势图 |
| S-08 | 查看专注记录 | 最近 50 条专注记录（日期 / 名称 / 时长 / 状态） |
| S-09 | 高效时段（时段分布） | 按小时统计专注时长，识别高效区间 |

#### 4.6.3 统计设置（齿轮入口）

| 设置项 | 说明 |
| --- | --- |
| 显示起床 / 睡眠打卡分布 | 开关控制对应区域显隐 |
| 显示月度打断原因分布 | 开关控制对应区域显隐 |
| 趋势图线条 | 曲线 / 直线 |
| 统计图单位 | 分钟 / 小时 |
| 月度数据展示范围 | 近 7 天 / 整月 |

### 4.8 未来计划表

#### 4.8.1 功能描述
记录生活中重要事项的日期与倒计时，即使事情结束也不会自动消失，保留作为纪念。

## 五、数据库设计

### 5.1 设计说明
- **双后端存储**：默认使用 **SQLite**（`%USERPROFILE%\.qingning_todo\qingning_todo.db`），无需安装数据库、开箱即用；可在设置中切换为 **MySQL**（`qingning_todo` 库，utf8mb4 / InnoDB）以实现多设备同步。DAO 层通过统一的 `Database` 接口透明访问两种后端，SQLite 后端会自动将 MySQL 风格 SQL 转换为 SQLite 兼容语法。
- 本地单用户模式，默认单用户；保留 `user` 表以支持多本地档案（可选）。
- 所有时间字段使用 `DATETIME`，时长使用秒（`INT`）为单位。
- 连接信息（数据库后端选择 + MySQL 主机 / 端口 / 用户 / 密码 / 库名）保存在用户主目录下 `config.json`（可用环境变量 `QINGNING_TODO_HOME` 覆盖目录，便于测试隔离）。
- 首次启动使用 SQLite 时无需任何配置，自动建库建表并播种默认用户 / 白噪音 / 成就 / 设置；仅当切换为 MySQL 模式且尚未配置时，才弹出「数据库连接配置」对话框（可测试连接、保存并连接、自动建库建表）。

## 九、附录：实现基准与版本说明

- 实现基准：以「番茄工作法（Pomodoro Technique）」这一公开的时间管理方法为功能范式，结合 Windows 桌面使用习惯独立设计与实现；产品定名为 **青柠待办**，主视觉采用青柠绿。
- 技术实现：Python 3 + PyQt6 桌面客户端；**本地存储双后端**——默认 SQLite（开箱即用，文件位于 `%USERPROFILE%\.qingning_todo\qingning_todo.db`），可选 MySQL（设置中切换，适合多设备同步）；图表由 `QPainter` 自绘、图标为内联 SVG，零图片素材；白噪音为内置真实 WAV 素材（自然音 / 氛围音 / 轻音乐 三大类，约 38 种），通过 `winsound` 原生循环播放（非 Windows 平台优雅降级），自定义上传自动转码为 WAV。
- 本版（v1.3）相对原始规格的偏差已在第三节「功能范围说明」中逐条标注：学霸模式 / 定时锁屏仅完成配置层，定目标 / 成就徽章展示 / 悬浮窗 / 音量控制 / 学习心得 / 导入恢复等为占位或规划中；早起鸟徽章为数据占位、未参与自动评估。
- v1.3 窗口交互优化（相对 v1.2）：① 关闭窗口改为「确认框」交互（退出 / 最小化到托盘 / 取消），不再默认直接最小化到托盘；② 任务栏图标点击可在「显示 ↔ 收起」间切换；③ 收起 / 展开采用几何缩放平滑动画，并禁用该窗口的 DWM 最小化过渡动画以消除收起时的闪烁；④ 系统托盘仅在可用时创建，避免虚拟桌面 / 远程会话下弹出 fallback 小窗口。

> 说明：本文档为面向「当前实现」的产品使用规格说明书，用于指导青柠待办（Windows 本地版，PyQt6 + SQLite/MySQL）的设计、开发与验收；功能以代码实际交付为准，规划中能力将随版本迭代补充。
