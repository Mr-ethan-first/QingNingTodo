# 青柠待办 · Windows 本地版

一款基于「番茄工作法（Pomodoro Technique）」的本地时间管理与专注效率工具，
核心理念为 **规划 → 追踪 → 记录 → 复盘**。
使用 **Python 3 + PyQt6** 构建 Win11 Fluent 风格桌面界面，
**本地 SQLite（默认，开箱即用）或 MySQL（可选，适合多设备同步）** 存储全部数据，
无广告、秒启动、纯本地运行。

设计语言「一抹青柠」：以清新青柠绿（`#7CB342` 浅色 / `#AED581` 深色）为主色，
大圆角卡片 + 分层投影、半透明玻璃质感侧栏。提供 **两套主题**，并支持自定义主色调，
可在「设置 → 外观主题」中一键切换：

- **皓白 · 清新白**：清爽明亮、留白充足的现代浅色风格
- **墨夜 · 深邃黑**：沉稳高级、低饱和的深色风格

## 功能一览

| 模块 | 说明 |
| --- | --- |
| 待办清单 | 待办集分组、4 种计时方式（普通番茄钟 / 正计时 / 倒计时 / 严格模式）、养习惯类型 + 打卡、优先级 / 重复 / 提醒、拖拽排序、单待办更多设置 |
| 番茄钟 | 开始 / 暂停 / 继续 / 放弃 / 提取完成、番茄钟循环、休息前询问、倒计时自动转正计时、暂停超时自动放弃、午夜模式、白噪音 |
| 数据统计 | 概览卡片、时长分布（柱 + 饼）、时段热力图、月度趋势、起床 / 睡眠分布、月度打断原因、年度数据、查看专注记录 + 统计设置 |
| 未来计划表 | 重要事项倒计时，过期不消失作为纪念 |
| 设置 | 昵称、界面主题（一键换肤）+ 主题色、专注计时参数、待办显示、习惯提醒、系统集成、习惯养成、定时锁屏配置、本地数据管理、数据库后端切换 |

> 已排除云同步 / 云分享 / 番茄自习室等联网社交与云功能（可选 MySQL 仅用于本地多设备同步）。
> 学霸模式与定时锁屏当前仅提供配置 / 数据管理界面，系统级拦截 / 锁屏行为尚未接入（详见规格说明书第三节）。

## 环境要求

- Windows 10 / 11 (x64)
- Python 3.10+（开发运行）
- （可选）MySQL 5.7 / 8.0，仅当在设置中切换为 MySQL 后端时需要

## 快速开始（源码运行）

```powershell
pip install -r requirements.txt
python -m src.main
```

**默认使用 SQLite，无需任何数据库配置即可启动**，程序自动在
`%USERPROFILE%\.qingning_todo\qingning_todo.db` 建库建表并播种默认数据。

如需使用 MySQL（适合多设备同步），在「设置 → 本地数据 → 数据库后端」中切换为 MySQL，
填写连接信息并测试连接后重启生效。连接信息保存在 `%USERPROFILE%\.qingning_todo\config.json`。

## 打包为 EXE

```powershell
python build.py
```

生成单文件 EXE `dist\青柠待办.exe`（PyInstaller onefile，无控制台窗口），
音频资源（`assets/sounds/`）与帮助文档自动打包进单文件。双击即可运行，默认 SQLite 开箱即用。

## 自动化测试

```powershell
python -m pytest
```

测试默认使用 **SQLite 后端**（无需 MySQL）；当检测到可用的 MySQL（`127.0.0.1:3306 root/123456`）
时自动启用 MySQL 集成测试，否则回退到 SQLite，保证无 MySQL 环境下测试不全部跳过。
测试通过环境变量 `QINGNING_TODO_HOME` 隔离数据目录，结束自动清理。

测试覆盖：应用名称、配置读写、连接与建表、种子数据幂等、全部 DAO 的 CRUD、
专注统计（累计 / 今日 / 热力图 / 高效时段 / 待办集分布 / 连续天数）、午夜模式归属、成就解锁、
焦点计时器、主题切换、内联图标、白噪音等。

## 技术要点

- **界面**：PyQt6 + 全局 QSS，Win11 Fluent 风格；图表由 `QPainter` 自绘、图标为内联 SVG，零图片素材，切换主题时配色统一重建。
- **数据存储**：双后端——默认 **SQLite**（文件即库，开箱即用），可选 **MySQL**（设置内切换，适合多设备同步）；DAO 层通过统一 `Database` 接口透明访问，SQLite 后端自动将 MySQL 风格 SQL 转译为 SQLite 语法；共 14 张表。
- **白噪音**：内置约 38 种真实 WAV 素材，分自然音 / 氛围音 / 轻音乐三类，由 `winsound` 原生循环播放；支持自定义上传（自动转码为 WAV）；播放器为进程级单例，避免孤儿音轨；非 Windows 平台优雅降级。
- **系统集成**：`QSharedMemory` 单实例、系统托盘（点击切换显示/收起、关闭时弹确认框可「最小化到托盘」）、任务栏图标点击切换显示↔收起、收起/展开带平滑缩放动画、右下角滑出式 Toast 通知、开机自启配置。
