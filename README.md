# 番茄 ToDo · Windows 本地版

基于《番茄ToDo-Windows本地版-产品使用规格说明书》复刻的桌面效率工具，
使用 **Flet（Flutter）** 构建现代化界面，**MySQL** 本地存储数据。
首次启动即可配置数据库连接并测试连通性。

界面提供 **两套精心设计的主题风格**，可在「设置 → 外观主题」中一键切换：

- **皓白 · 清新白**：清爽明亮、留白充足的现代浅色风格
- **墨夜 · 深邃黑**：沉稳高级、低饱和的深色风格

## 功能一览

| 模块 | 说明 |
| --- | --- |
| 待办清单 | 待办集分类、4 种计时方式（普通番茄钟/正计时/倒计时/严格模式）、时长/循环/优先级/重复 |
| 番茄钟 | 开始/暂停/放弃/提取完成、休息模式、番茄钟循环、白噪音选择 |
| 数据统计 | 累计/今日专注、连续天数、近30天热力图、高效时段、待办集分布、成就徽章、历史时间轴 |
| 未来计划表 | 重要事项倒计时，过期不消失作为纪念 |
| 设置 | 昵称、界面主题（一键换肤）、专注目标（日/周/月）、专注白名单、定时锁屏、数据备份、修改数据库连接 |

> 按规格要求，已排除云同步 / 云分享 / 番茄自习室等联网社交与云功能。

## 环境要求

- Windows 10 / 11 (x64)
- Python 3.10+（开发运行）
- MySQL 5.7 / 8.0（本地）

## 快速开始（源码运行）

```powershell
pip install -r requirements.txt
python -m src.main
```

首次启动会弹出「数据库连接配置」对话框：

1. 填写 主机 / 端口 / 用户名 / 密码 / 数据库名
2. 点击 **测试连接** 验证连通性
3. 点击 **保存并连接**，程序自动创建数据库与数据表

连接信息保存在 `%USERPROFILE%\.qingning_todo\config.json`。

## 打包为 EXE

```powershell
python build.py
```

生成 `dist\番茄ToDo\番茄ToDo.exe`（Flet 桌面应用，无控制台窗口）。
双击即可运行，运行时按上述流程配置数据库。

## 自动化测试

```powershell
python -m pytest
```

测试默认连接 `127.0.0.1:3306 root/123456`，使用独立测试库 `qingning_todo_pytest`，
测试结束自动清理。可用环境变量覆盖连接：

```powershell
$env:TEST_DB_USER="root"; $env:TEST_DB_PASSWORD="123456"; python -m pytest
```

若本地无可用 MySQL，数据库相关用例会自动跳过（配置类用例仍会运行）。

测试覆盖：配置读写、连接与建表、种子数据幂等、全部 DAO 的 CRUD、
专注统计（累计/今日/热力图/高效时段/待办集分布/连续天数）、午夜模式归属、成就解锁等。

## 目录结构

```
qingNingTodo/
├── src/
│   ├── main.py                # 应用入口（Flet 启动 + 数据库初始化）
│   ├── config.py              # 配置读写（本地 JSON）
│   ├── theme.py               # 主题令牌（白色 / 黑色两套）
│   ├── database/
│   │   ├── schema.py          # 12 张表 DDL 与种子数据
│   │   ├── connection.py      # 连接管理 + 测试连接 + 初始化
│   │   └── dao.py             # 各实体 DAO 与统计查询
│   └── ui_flet/
│       ├── state.py           # 全局状态（db / 主题 / 订阅）
│       ├── widgets.py         # 主题感知通用组件
│       ├── pad.py / align.py  # padding / alignment 兼容层
│       ├── main_view.py       # 主窗口（侧边导航 + 页面容器）
│       ├── db_config_view.py  # 数据库配置对话框（测试连接）
│       └── pages/             # 待办/专注/统计/计划/设置 页面
├── tests/                     # pytest 自动化测试
├── build.py                   # flet pack 打包脚本
├── requirements.txt
├── pytest.ini
└── 番茄ToDo-Windows本地版-产品使用规格说明书.md
```

## 说明

- 专注模式（学霸模式）与定时锁屏在规格中定义为桌面行为管控，本版本聚焦数据与计时核心，
  相关配置项（白名单、锁屏时段）已提供数据管理界面，实际系统级拦截可按需扩展。
